NOTE: Loki is configured for S3 storage. Ensure you set the S3 credentials:
  --set loki.s3.endpoint=<endpoint> \
  --set loki.s3.bucket=<bucket> \
  --set loki.s3.accessKeyId=<key> \
  --set loki.s3.secretAccessKey=<secret>

Or create an existing secret and reference it with:
  --set loki.existingSecret=<secret-name>
